# FitTrack

A personal workout tracker built to log sessions, follow training plans, track progress over time, and record personal bests.

Live site: https://workout-tracker-taupe-rho.vercel.app/
GitHub: https://github.com/gbariana2/workout-tracker

---

## What it does

FitTrack has four core sections:

- **Log** — record a workout session with exercises, sets, reps, and weight
- **Plans** — create and follow structured training templates
- **Progress** — visualise how your lifts and volume have changed over time
- **PRs** — a personal records board that tracks your best lifts across exercises

---

## Tech stack

- Next.js (TypeScript)
- CSS for styling
- Deployed on Vercel

---

## Running locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.
